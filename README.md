## Que es esto?

Este repositorio contiene la **guia de identidad de marca** de [Novaly X Cash](https://novalyxcash.com), la plataforma SaaS de cadenas de ahorro rotativo (ROSCA) digitales + comunidad financiera.

La guia se publica automaticamente en **GitHub Pages** cada vez que se actualiza la rama `main`.

## Contenido

| Seccion | Descripcion |
|---------|-------------|
| **01 — Identidad** | Mision, vision, arquitectura del nombre, taglines |
| **02 — Logo** | Uso del logo, reglas, versiones pendientes |
| **03 — Colores** | Paleta completa con primarios, variantes, neutros y gradientes |
| **04 — Tipografia** | Inter — escala tipografica y clases Tailwind |
| **05 — Tono de Voz** | Pilares de comunicacion, do's & don'ts, guia por canal |
| **06 — Aplicaciones** | Variables CSS y configuracion Tailwind |
| **07 — Social Media** | Formatos por plataforma, estilo de publicaciones, hashtags |
| **08 — Assets** | Logos descargables y estructura del repo |

## Paleta de colores

| Color | Hex | Uso |
|-------|-----|-----|
| Naranja | `#F5820A` | CTAs, botones, acentos, precios |
| Dorado | `#F5C518` | Highlights, badges premium, gradientes |
| Morado | `#7B4FD4` | Plan Ultra, elementos premium |
| Navy | `#1A1F2E` | Fondos principales, headers |
| Blanco | `#FFFFFF` | Texto sobre fondos oscuros |

## Tipografia

**[Inter](https://rsms.me/inter/)** — la unica tipografia de la marca. Pesos: 300–900.

## Desarrollo local

Abri `index.html` en tu navegador. No necesita build, bundler, ni servidor — es HTML puro.

```bash
open index.html
```

## Deploy

El sitio se despliega automaticamente a [GitHub Pages](https://novalyxcash.github.io/brand) via GitHub Actions cada vez que se pushea a `main`.
